GitHub Actions 可以帮助你自动化构建和部署 Jekyll 博客到 GitHub Pages。

## 工作流配置

在你的仓库中创建 `.github/workflows/pages.yml`，即可启用自动部署。

## 优点

1. 推送代码即自动部署
2. 无需本地安装 Jekyll 环境
3. 支持自定义构建步骤

开始自动化你的博客部署吧！
